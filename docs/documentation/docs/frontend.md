The frontend is a modern single-page application (SPA) built with **Vue.js 3** and **Vite**.

### Tech Stack
- **Framework**: Vue.js 3
- **Build Tool**: Vite
- **Styling**: Vanilla CSS
- **Deployment**: Nginx (via Docker)

### Features
- **Real-time Monitoring**: Visualizes data captured by the packet sniffer and stored in the database.
- **Dashboard**: Interfaces with the backend API to display device status and logs.
- **Responsive Design**: Optimized for various screen sizes using modern CSS.
